# telegram-agent-helix-bot

A personal Telegram AI agent for two — built with Claude, Pinecone memory, and Tavily search. Set in Berlin.

- **Run the bot (refactored):** `python app/main.py` (project root, venv on; see `.env.example`).
- **Learning materials:** `learning_modules/` (stage scripts, `PLAN.md`, workbooks including `stage_workbooks/stage_refactor.ipynb`).
